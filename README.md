# FFXIV BLM Full BiS Solver

Python 3.10.11

This repository provides a fully-featured BiS gear solver for Black Mage in FFXIV. Features include:

- Rotation-aware DPS simulation
- Full stat formulas (Crit, DH, Det, SpellSpeed)
- Materia optimization and overmeld handling
- Slot-based combinator with top-N BiS output
- XIVAPI loader with caching and robots.txt compliance
- Tkinter GUI interface
- Logging of all operations
